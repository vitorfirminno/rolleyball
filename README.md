# rolleyball

Projeto final (desenvolvimento de uma landing page) para uma empresa.

Fizemos o projeto inspirado na empresa local de quadras chamada RolleyBall, onde observamos que eles não tinham site, apenas medias sociais como o Instagram e WhatsApp.
Tentamos contato com o dono e apresentamos nosso projeto, conseguimos apenas autorização para apresentar em sala de aula.
